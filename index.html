About Us:

Learnify is an interactive Learning Management System (LMS) website designed to make learning engaging, organized, and rewarding for students. It provides a digital platform where users can register or log in, access sample courses, and track their learning progress through animated progress bars and smooth transitions. Built using HTML, CSS, JavaScript, while JavaScript ensures proper form validation. The dashboard includes four main pages—Home, Courses, Badges, and Announcements. The Courses page features three sample courses with topics, quizzes, and completion buttons that dynamically update progress bars, while the Badges section rewards learners for completing content to encourage consistent participation. Additional features include a “Quote of the Day” or “Tip of the Week” and a scrolling Announcements Ticker displaying updates and messages. Overall, Learnify showcases the essential features of an LMS in a visually appealing way, combining interactivity, creativity, and technology to make learning smarter and more enjoyable.
